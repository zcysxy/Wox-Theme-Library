## How to Apply

Just copy the `*.xaml` theme file(s) to your Wox's `Themes` directory.

```shell
C:\Users\user_name\AppData\Local\Wox\app-1.1.1111\Themes\FluentLight.xaml
```

> Change `user_name`, `1.1.1111` accordingly.
